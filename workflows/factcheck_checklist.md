# Factcheck Checklist

원고 완성 후, 디자인 전에 반드시 실행.
모든 항목 통과해야 다음 단계 진행.

---

## 정보 검증

### 출처
- [ ] 공식 출처가 있는가? (개인 블로그, 뉴스 기사 단독 출처 불가)
- [ ] 출처 URL이 현재 접근 가능한가?
- [ ] 출처가 정부/기관/브랜드 공식 채널인가?
- [ ] 확인일이 오늘 날짜 or 이번 주 이내인가?

### 유효성
- [ ] 마감일이 아직 지나지 않았는가?
- [ ] 신청/참여가 현재 가능한가?
- [ ] 이미 마감된 행사를 현재형으로 쓰고 있지 않은가?

### 대상 조건
- [ ] 나이 조건이 명확하게 표기되어 있는가?
- [ ] 거주지 조건이 있으면 카드뉴스에 명시되어 있는가?
- [ ] 소득 조건이 있으면 카드뉴스에 명시되어 있는가?
- [ ] 재학/취업 여부 조건이 있으면 명시되어 있는가?

### 비용
- [ ] 비용 정보가 있는가?
- [ ] "무료"라고 표기했는데 실제로 무료인가?
- [ ] 유료 전환 가능성이 있으면 주의사항에 포함되어 있는가?

### 신청 방법
- [ ] 신청 방법이 구체적으로 표기되어 있는가?
- [ ] 신청 링크/앱/위치가 현재 유효한가?

---

## 지역/타겟 검증

- [ ] 서울/수도권 대상이 맞는가?
- [ ] 지방 한정 혜택을 서울 계정에서 다루고 있지 않은가?
- [ ] 20대 대상이 맞는가? (조건이 너무 좁지 않은가?)

---

## 브랜드 검증

- [ ] 얇은지갑클럽 톤인가? (`brand/voice_and_tone.md` 참조)
- [ ] `brand/forbidden_rules.md`에 나열된 금지 표현이 없는가?
- [ ] 공공기관 말투가 없는가?
- [ ] 과장 표현이 없는가?
- [ ] 광고성 콘텐츠인데 표시가 없는 경우가 없는가?
