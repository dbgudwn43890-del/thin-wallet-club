# Ideas Backlog Index

얇은지갑클럽의 아이디어 백로그 인덱스.
이 파일은 실제 후보 데이터 파일이 아니다.

현재 상태: **구조 세팅 단계**
확인일: 2026.06.23

---

## 역할

`content/ideas.md`는 운영자가 앞으로 다룰 수 있는 주제 방향, 반복 포맷, 리서치 질문을 모아두는 인덱스다.
실제 정책, 행사, 전시, 팝업, 브랜드 프로그램 후보는 이 파일에 저장하지 않는다.

실제 후보 데이터는 반드시 아래 위치에 저장한다.

```
data/candidates/{category}_candidates.yml
```

검증을 통과한 후보는 아래 위치로 이동한다.

```
data/verified/{id}_{slug}.yml
```

폐기된 후보는 아래 위치에 사유와 함께 보존한다.

```
data/rejected/{date_or_slug}.yml
```

---

## 저장 금지 항목

이 파일에는 다음을 쓰지 않는다.

- 실제 정책명
- 실제 전시명
- 실제 행사명
- 실제 팝업명
- 실제 브랜드 프로그램명
- 실제 신청 링크
- 실제 마감일
- 검증 전 공식 출처 URL
- 큐레이터 점수
- 제작 확정 표시

---

## 백로그 항목 형식

실제 후보가 아니라 방향성만 적는다.

```markdown
### {아이디어 방향}

- 카테고리:
- 독자 문제:
- 리서치 질문:
- 후보 데이터 저장 위치: data/candidates/{category}_candidates.yml
- 제작 전 필요한 확인:
- 상태: backlog / research_needed / paused
```

## 예시

아래 항목은 실제 후보가 아니라 작성 형식 예시다.

### 예시: 무료 또는 저가 문화생활 묶음

- 카테고리: culture
- 독자 문제: 주말에 돈을 많이 쓰지 않고도 취향 있어 보이는 경험을 찾고 싶다.
- 리서치 질문: 서울/수도권에서 공식 출처로 확인 가능한 무료 또는 3만 원 이하 문화 프로그램이 있는가?
- 후보 데이터 저장 위치: `data/candidates/culture_candidates.yml`
- 제작 전 필요한 확인: 공식 출처, 운영 기간, 비용, 예약 필요 여부, 지역 조건
- 상태: backlog

### 예시: 청년 지원 제도 압축 정리

- 카테고리: policy
- 독자 문제: 내가 받을 수 있는 지원이 있는지 빠르게 알고 싶다.
- 리서치 질문: 서울/수도권 20대에게 실질적으로 도움이 되는 공식 청년정책이 있는가?
- 후보 데이터 저장 위치: `data/candidates/policy_candidates.yml`
- 제작 전 필요한 확인: 공식 출처, 신청 대상, 신청 기간, 지원 금액, 제외 조건
- 상태: backlog

### 예시: 대학생 전용 혜택 정리

- 카테고리: student
- 독자 문제: 학교 이메일이나 재학 상태로 받을 수 있는 혜택을 놓치고 싶지 않다.
- 리서치 질문: 대학생만 사용할 수 있고 공식 페이지에서 조건 확인이 가능한 혜택이 있는가?
- 후보 데이터 저장 위치: `data/candidates/student_candidates.yml`
- 제작 전 필요한 확인: 공식 출처, 재학 인증 방식, 무료/할인 조건, 사용 제한
- 상태: backlog

## 제작 확정 전 이동 금지

제작 확정 전에는 어떤 파일도 `drafts/`로 이동하지 않는다.

`drafts/`는 Curator가 `selected` 상태로 확정한 콘텐츠만 사용한다.
아이디어 단계와 검증 단계의 파일은 `data/` 아래에서만 관리한다.

---

## 다음 단계 진입 조건

아이디어가 실제 후보 데이터가 되려면 다음 조건을 충족해야 한다.

- Scout가 공식 출처 후보를 찾는다.
- `schemas/candidate.schema.yml` 형식으로 `data/candidates/`에 저장한다.
- Verifier가 `ops/source_rules.md` 기준으로 검증한다.
- Curator가 점수화하고 제작 여부를 결정한다.

이 조건 전에는 카드뉴스 원고, 캡션, 디자인 프롬프트를 만들지 않는다.
