# Design System

## Brand

얇은지갑클럽

## Format

Instagram carousel, 1080x1350px.

## Mood

Magazine, editorial, urban, minimal, warm, curated.

## Visual principles

1. 정보는 명확하게.
2. 분위기는 세련되게.
3. 여백을 넉넉하게.
4. 한 장에는 하나의 메시지만.
5. 숫자, 금액, 마감일은 크게.
6. 출처와 확인일은 하단에 작게 (필수).
7. 아이콘과 장식은 최소화.
8. 공공기관 카드뉴스처럼 보이지 않게.

## Slide system (8장 기준)

| 슬라이드 | 역할 | 핵심 요소 |
|---------|------|-----------|
| 1 | Cover / Hook | 제목, 카테고리 라벨, 브랜드명 |
| 2 | Context | 이게 뭔데? |
| 3 | Why now | 왜 지금 봐야 해? |
| 4 | Target | 누구에게 해당하는가? |
| 5 | Benefit | 얼마나 이득인가? |
| 6 | How-to | 신청/이용 방법 |
| 7 | Caution | 주의사항, 추천 대상 |
| 8 | CTA + Source | 저장/공유 CTA + 출처 + 확인일 |

## Fixed footer (정보성 카드 전체 필수)

```
확인일: YYYY.MM.DD
출처: [공식 출처명]
```

## Typography direction

- 제목: 굵고 짧게
- 본문: 얇고 가독성 좋게
- 강조: 볼드 or 컬러 포인트
- 글자 크기 위계 3단계 이상 유지

## Color direction

- 배경: 크림, 오프화이트, 미드그레이 계열
- 포인트: 계절/주제에 따라 변주
- 텍스트: 다크 그레이 (순수 검정 지양)
- 공공기관 느낌 나는 파랑/빨강 원색 금지

## Cover 슬라이드 구성

```
[카테고리 라벨 — 소문자 or 영문]
[메인 제목 — 1~2줄, 굵게]
[서브 제목 or 날짜/기간 — 선택]
[얇은지갑클럽]
```

커버 문구는 명확한 정보 중심으로 구성한다.
감도는 "쿨한 척하는 카피"가 아니라 타이포그래피, 여백, 정보 위계로 만든다.
브랜드명성 문구나 내부 기획어를 커버에 넣지 않는다.

권장 커버 방향:
- 서울 무료 전시 5
- 지금 볼 수 있는 무료 전시
- 주말에 가기 좋은 무료 전시 5
- 이번 주말 가볼 만한 곳
- 저장해둘 서울 전시

금지 커버 방향:
- 무료인데 꽤 괜찮아
- 얇은지갑클럽 저장용 리스트
- 감성 무료 전시
- 요즘 핫한 전시
- 힙한 무료 전시

## CTA 슬라이드 구성 (8장)

```
[저장해두고 필요할 때 꺼내봐. or 비슷한 CTA]
[팔로우하면 이런 정보 계속 볼 수 있어.]

출처: [출처명]
확인일: YYYY.MM.DD
얇은지갑클럽
```

## 디자인 금지 목록

- 무지개 그라데이션
- 밈 스티커, 이모지 과용
- 복잡한 배경 패턴
- 정보 카드에 배경 이미지 (텍스트 가독성 저하)
- 출처/확인일 누락
- 한 장에 5줄 이상 본문
- 내부 기획어를 독자-facing 문구처럼 크게 노출
- 브랜드명을 CTA 외 영역에서 과하게 반복
