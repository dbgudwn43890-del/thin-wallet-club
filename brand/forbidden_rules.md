# Forbidden Rules

얇은지갑클럽에서 절대 해서는 안 되는 것들.
이 규칙을 어기면 브랜드가 망가진다.

---

## 정보 관련

### 절대 금지

- 공식 출처 없이 정보 발행
- 마감일이 지난 혜택 발행
- 서울/수도권 외 지역 정책을 서울 계정인 것처럼 발행
- 대상 조건 (나이, 소득, 거주지, 재학) 생략
- 확인일 없이 정보 발행
- 비용 정보 누락 or 불명확하게 표기

### 조건부 금지

- 출처가 뉴스 기사인 경우 → 공식 링크 추가 확인 필수
- 브랜드 프로그램이 유료로 전환된 경우 → 발행 중단
- 신청 링크가 없거나 접근 불가인 경우 → 검증 후 재확인

## 톤/카피 관련

### 절대 금지

- "대박", "역대급", "무조건", "꼭 받으세요" — 과장 표현
- "청년 여러분", "신청하시기 바랍니다" — 행정 말투
- "이거 모르면 손해", "알고 보면 다 알아" — 공포 마케팅
- "우리 팔로워 분들" — 공공기관 SNS 말투
- "무료인데 꽤 괜찮아" — 쿨해 보이려는 말투
- "얇은지갑클럽 저장용 리스트" — 내부 기획어 노출
- "감성 무료 전시" — 분위기를 정보보다 앞세우는 표현
- "요즘 핫한", "힙한", "MZ 취향" — 유행어로 감도를 흉내 내는 표현
- "안 가면 손해", "무조건 가야 함", "모르면 손해" — 과장된 추천/공포 마케팅
- 영어 남용 (불필요한 영단어 혼용)
- 문장 끝에 "^^", "~" — 구어체 과잉

### 주의 필요

- 1인칭 "나는" 사용 — 에세이형 외 사용 자제
- 명령형 — "하세요" 대신 "하면 좋다" 권장
- 물음표 — 한 카드에 1개 이하
- 브랜드가 자기 감도를 과하게 의식하는 말투
- 내부 운영어를 독자-facing 문구처럼 쓰는 방식
- 정보보다 분위기를 앞세우는 말투

## 디자인 관련

### 절대 금지

- 출처, 확인일 누락
- 한 슬라이드에 정보 5줄 초과
- 공공기관 색상 (원색 파랑/빨강)
- 밈 스티커, 유행 이모지 남용
- 1080x1350 비율 이탈
- 브랜드명 과다 노출
- 커버에서 내부 기획어 사용

### 주의 필요

- 이미지 배경 위에 텍스트 (가독성 확인 필수)
- 그라데이션 (단순하게)
- 폰트 3종 초과 사용

## 콘텐츠 방향 관련

- 정치적 의견 개입 금지
- 특정 브랜드 유료 광고를 자연스러운 정보처럼 발행 금지
- 광고성 콘텐츠는 반드시 #광고 #협찬 명시
- 타 계정 콘텐츠 무단 복제 금지
- 대학생 콘텐츠 비중이 30% 초과되지 않도록 유지
