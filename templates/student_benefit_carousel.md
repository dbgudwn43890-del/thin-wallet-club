# Student Benefit Carousel Template

대학생 전용 혜택, 학생 할인, 무료 툴, 대외활동 콘텐츠용 템플릿.
전체 콘텐츠의 10~20% 이내로 유지. 격주 1회 권장.

---

## Metadata (필수)

```
콘텐츠명:
카테고리: 대학생이면 저장
확인일: YYYY.MM.DD
공식 출처:
출처 URL:
대상: (대학생/대학원생/특정 학년)
지역: (전국/서울/온라인)
기간: (상시/기간 한정)
마감일: (있으면)
비용: (무료/할인가)
혜택:
```

---

## Slide 1: Hook (커버)

```
[카테고리 라벨: 대학생 혜택]

[제목: 15자 이내]
예시: "대학 이메일로 무료인 것들"
예시: "재학 중에만 받을 수 있는 것"

얇은지갑클럽
```

---

## Slide 2: Why now — 재학 중에만 가능한 이유

```
[제목: "재학생이라서 할 수 있는 것"]

[본문: 2줄]
졸업하면 못 받는 혜택임을 명확히.
시간은 지나고 있다는 걸 느끼게.

확인일: YYYY.MM.DD | 출처: [출처명]
```

## 콘텐츠 아이디어 예시

- 대학 이메일로 무료인 소프트웨어 (Notion, Figma, GitHub Pro 등)
- 방학 전 신청해야 하는 무료 프로그램
- 대학생 전시/공연 할인 모음
- 대학생 포트폴리오에 도움 되는 무료 사이트
- 학생증으로 받을 수 있는 할인 모음
- 대학생 공모전 상금 높은 것 모음
