# Brand Program Carousel Template

브랜드 프로그램, 공모전, 체험 기회, 콜라보 콘텐츠용 템플릿.

---

## Metadata (필수)

```
콘텐츠명:
카테고리: 브랜드가 주는 기회
확인일: YYYY.MM.DD
공식 출처:
출처 URL:
브랜드명:
프로그램 유형: (공모전/체험/교육/인턴십/팝업)
지역: (서울/온라인/전국)
대상: (나이/재학/직업 조건)
마감일:
비용: (무료/유료)
혜택:
```

---

## Slide 1: Hook (커버)

```
[카테고리 라벨: 브랜드 기회]

[제목: 브랜드명 + 무엇을 주는지]
예시: "현대카드가 무료로 여는 공연"
예시: "무신사가 20대한테 주는 기회"

얇은지갑클럽
```

---

## Slide 2: What — 이 프로그램 뭐야?

```
[제목: 프로그램명]

[본문: 2~3줄]
어떤 브랜드가, 무엇을, 왜 주는지.
마케팅 언어 제거하고 실질 내용만.

확인일: YYYY.MM.DD | 출처: [출처명]
```

---

## Slide 3: Why — 왜 이게 좋은 기회야?

```
[제목: 기회의 핵심]
예시: "포트폴리오 + 상금 + 브랜드 연결"

[본문: 2~3줄]
단순 혜택 나열 말고, "이 기회가 왜 특별한지" 설명.

확인일: YYYY.MM.DD | 출처: [출처명]
```

---

## Slide 4: Target — 누가 지원할 수 있어?

```
[제목: "이런 사람이면 지원 가능"]

[조건: 3~5개]
- 나이/재학 여부
- 필요 스킬/경력
- 지역 조건
- 팀/개인 여부

확인일: YYYY.MM.DD | 출처: [출처명]
```

---

## Slide 5: Benefit — 실제로 뭘 받아?

```
[제목: 구체적 혜택]
예시: "상금 XXX만 원 + 전시 기회 + 포트폴리오"

[본문: 혜택 목록]

확인일: YYYY.MM.DD | 출처: [출처명]
```

---

## Slide 6: How to Apply

```
[제목: "지원 방법"]

[단계: 3~4단계]
1. 어디서 신청하는지
2. 필요한 것
3. 일정

확인일: YYYY.MM.DD | 출처: [출처명]
```

---

## Slide 7: Caution + Recommendation

```
[제목: "이건 확인해"]

[주의사항: 2~3개]
- 실제로 돈이 드는 부분 (있다면)
- 저작권/소유권 관련 (있다면)
- 이런 사람한테 특히 맞음

확인일: YYYY.MM.DD | 출처: [출처명]
```

---

## Slide 8: CTA + Source

```
저장해두고 마감 전에 꼭 확인해.

팔로우하면 이런 기회 계속 올려줄게.

출처: [공식 출처명]
확인일: YYYY.MM.DD
얇은지갑클럽
```

## Caption 템플릿

```
[첫 줄 — 브랜드 + 기회 요약]
예시: [브랜드명]에서 20대한테 이런 기회 줌.

[핵심 정보]
마감: YYYY.MM.DD
대상: [조건 한 줄]
혜택: [핵심 혜택]

[CTA]

출처: [출처명] | 확인일: YYYY.MM.DD

#얇은지갑클럽 #브랜드기회 #20대기회 ...
```
