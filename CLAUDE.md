# 얇은지갑클럽 Project Instructions

얇은지갑클럽은 서울/수도권 20대를 위한 감각 있는 생활 정보 매거진이다.

## Positioning

돈은 많지 않지만 감각 있게 살고 싶은 20대를 위해 저렴한 문화생활, 청년 혜택, 브랜드 프로그램, 대학생 혜택, 취향 입문 정보를 큐레이션한다.

## Target

- 서울/수도권 중심
- 20대 전체
- 대학생 콘텐츠는 전체의 10~20% 비중
- 돈은 아끼고 싶지만 촌스럽게 살고 싶지 않은 사람

## Tone

- 정보형이지만 감각 있게
- 친구 같은 말투보다 매거진 에디터 톤
- 과장 금지
- 행정용어를 쉽게 번역
- 공공기관 카드뉴스처럼 쓰지 않기
- 밈 계정처럼 가볍게 쓰지 않기

## Required for all factual content

모든 정책, 행사, 지원금, 브랜드 프로그램 콘텐츠에는 다음 정보를 반드시 포함한다.

- 확인일
- 공식 출처
- 지역
- 대상
- 기간 또는 마감일
- 비용
- 신청/이용 방법
- 주의사항

## Content pillars

1. 이번 주 느좋 일정
2. 청년 혜택 압축정리
3. 브랜드가 주는 기회
4. 돈 적게 드는 취향 입문
5. 대학생이면 저장
6. 돈과 취향에 대한 짧은 조언

## Design

- 1080x1350 인스타그램 카드뉴스
- 매거진형 디자인
- 여백 넉넉하게
- 제목은 짧고 강하게
- 숫자, 금액, 마감일 강조
- 출처와 확인일은 작게 고정
- 정보 과밀 금지

## Agent team

에이전트 역할 정의는 AGENTS.md 참조.
에이전트 파일은 .claude/agents/ 에 위치.

## File structure

```
thin-wallet-club/
  CLAUDE.md          ← 프로젝트 메모리 (이 파일)
  AGENTS.md          ← 에이전트 팀 정의
  brand/             ← 브랜드 전략, 톤, 디자인 시스템, 금지 규칙
  content/           ← 콘텐츠 캘린더, 후보, 발행 로그
  sources/           ← 공식 출처 목록 (yml)
  workflows/         ← 주간 파이프라인, 팩트체크, 검수 체크리스트
  templates/         ← 카드뉴스 템플릿 5종
  drafts/            ← 브리프/스크립트/디자인프롬프트/캡션
  outputs/           ← 검수 완료 및 발행 완료 파일
  .claude/agents/    ← 에이전트 파일 7개
  .claude/skills/    ← 스킬 파일 3종
```
