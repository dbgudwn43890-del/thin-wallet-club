# Upload Rules

인스타그램 업로드 전/후 운영 규칙.

---

## 업로드 전 필수 확인

`workflows/cardnews_checklist.md` 전체 통과가 선행 조건.

추가로 확인:

- [ ] 예약 업로드 시간 설정 (권장: 오전 7시 / 오후 7시)
- [ ] 슬라이드 순서 최종 확인 (1→8장)
- [ ] 캡션 복사 후 글자 수 확인 (인스타 최대 2,200자)
- [ ] 해시태그 15개 이내 확인
- [ ] 첫 댓글 문구 준비 완료

---

## 업로드 설정

| 항목 | 설정 |
|------|------|
| 포맷 | 캐러셀 (릴스 아님) |
| 위치 태그 | 서울 (or 해당 지역명) |
| 공동 작성자 | 없음 (초기) |
| 링크 | 프로필 링크 or 링크트리 |

---

## 첫 댓글 고정

업로드 직후 댓글란에 고정.

```
얇은지갑클럽은 서울/수도권 20대를 위한 생활 정보 매거진입니다.
무료·저가 문화생활, 청년 혜택, 브랜드 프로그램, 취향 입문 정보를 정리합니다.
저장해두면 나중에 필요할 때 찾기 편해요.
```

---

## 하이라이트 카테고리

| 카테고리 이름 | 포함 콘텐츠 |
|------------|-----------|
| 문화생활 | 전시/공연/팝업 |
| 청년혜택 | 지원금/정책/패스 |
| 브랜드기회 | 브랜드 프로그램/공모전 |
| 대학생저장 | 대학생 전용 혜택 |
| 짧은조언 | 에세이/관점형 콘텐츠 |

---

## 업로드 후 즉시

- [ ] `content/published_log.md`에 기록
  - 업로드일, 링크, 출처, 확인일
- [ ] 파일 → `outputs/published/`로 이동
- [ ] 첫 30분 댓글 확인 (질문 있으면 즉시 답변)

---

## 오류 발생 시

| 오류 유형 | 대응 |
|---------|------|
| 정보 오류 (경미) | 캡션 수정 + 스토리 정정 공지 |
| 정보 오류 (심각) | 즉시 삭제 + 정정 게시물 업로드 |
| 디자인 오류 | 24시간 이내 수정 재업로드 |
| 출처/확인일 누락 | 즉시 캡션 추가 수정 |

---

## 업로드 금지 조건

- 마감일이 3일 이내로 남은 정보 (리드타임 부족)
- Verifier 검증 미통과 상태
- 출처 URL 접근 불가 상태
- 비용 정보가 확인되지 않은 상태
