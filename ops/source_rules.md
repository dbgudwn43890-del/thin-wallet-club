# Source Rules

얇은지갑클럽이 콘텐츠에 사용할 수 있는 출처의 기준과 등급.

---

## 출처 등급

### Tier 1 — 1차 공식 출처 (사용 가능)

정부, 지자체, 공공기관, 브랜드의 공식 채널.

- 정부/지자체 공식 홈페이지 (gov.kr, seoul.go.kr 등)
- 공공기관 공식 홈페이지 (mmca.go.kr, sac.or.kr 등)
- 브랜드 공식 홈페이지
- 브랜드 공식 SNS (브랜드 계정 직접 운영)
- 공식 보도자료

### Tier 2 — 2차 출처 (교차 확인 필수)

공신력 있는 매체나 플랫폼. Tier 1 출처와 반드시 교차 확인 후 사용.

- 주요 언론사 보도 (조선, 중앙, 한겨레, 경향, 연합뉴스 등)
- 정부/기관 공식 블로그 (네이버 공식 블로그)
- 공공 데이터 포털

### Tier 3 — 참고만 가능 (단독 출처 불가)

정보 발굴 용도로만 활용. 반드시 Tier 1 출처를 직접 확인해야 함.

- 개인 블로그
- 유튜브 영상
- 인스타그램 게시물 (타 계정)
- 커뮤니티 게시글 (에브리타임, 블라인드, 레딧 등)
- 카카오채널/문자 안내

### 사용 불가

- 출처 표기 없는 캡처 이미지
- "~라고 들었다" 류의 간접 정보
- 이미 삭제된 페이지의 캐시
- 기간이 지난 공고문

---

## 출처 확인 절차

1. Tier 1 URL을 직접 접속하여 정보 확인
2. 접속 불가 시 검색으로 최신 공식 URL 재확인
3. 정보가 일치하지 않는 경우 최신 날짜 기준 우선
4. URL을 확인한 날짜를 `확인일`로 기록

---

## 출처 표기 형식

모든 콘텐츠에 다음 두 가지를 반드시 포함한다.

```
출처: [기관/브랜드명]
확인일: YYYY.MM.DD
```

URL은 캡션 또는 별도 링크로 안내한다. 카드뉴스 이미지 안에는 기관명만 표기.

---

## 출처 갱신 주기

| 콘텐츠 유형 | 재확인 기준 |
|-----------|-----------|
| 정책/지원금 | 발행 전 48시간 이내 재확인 |
| 문화행사/전시 | 발행 전 24시간 이내 재확인 |
| 브랜드 프로그램 | 발행 전 48시간 이내 재확인 |
| 상시 혜택 | 2주 이상 경과 시 재확인 |

`sources/policy_sources.yml`
`sources/culture_sources.yml`
`sources/brand_sources.yml`
`sources/university_sources.yml`
