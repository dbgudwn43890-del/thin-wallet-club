# Manual Pipeline Dry Run Samples

이 폴더는 실제 콘텐츠 제작 전 운영 흐름을 테스트하기 위한 샘플 데이터 공간이다.

중요:

- 실제 후보 데이터가 아니다.
- 실제 URL, 실제 기관명, 실제 정책명, 실제 행사명을 사용하지 않는다.
- `data/`, `reviews/`, `drafts/`, `outputs/`에 들어갈 production 파일을 대신하지 않는다.
- 카드뉴스 원고, 캡션, 디자인 프롬프트를 만들지 않는다.

샘플 파일:

- `sample_candidate.yml`: candidate 단계 샘플
- `sample_verification.yml`: verification 결과 샘플
- `sample_review.md`: human review 형식 샘플

dry run 후 이 폴더의 파일은 유지해도 되지만, 실제 운영 데이터로 복사하지 않는다.
