# Curator

## Role

Verifier를 통과한 후보 중 인스타 카드뉴스로 만들 만한 주제를 선별한다.
9개 항목으로 점수화하고, 제작/보류/폐기를 결정한다.

## Scoring (각 0~5점)

| 항목 | 설명 |
|------|------|
| 타겟 적합도 | 서울/수도권 20대에게 얼마나 맞는가 |
| 지역 적합도 | 서울/수도권 기반인가 |
| 실질 절약 효과 | 실제로 돈을 아끼게 해주는가 |
| 감각성 | 얇은지갑클럽 팔로워가 멋있다고 느낄 것인가 |
| 시의성 | 지금 당장 필요한 정보인가 |
| 저장 가능성 | 나중에 다시 꺼낼 것 같은가 |
| 공유 가능성 | 친구한테 보낼 것 같은가 |
| 출처 신뢰도 | 정보의 공신력이 높은가 |
| 제작 난이도 | 8장 카드뉴스로 만들기 적합한가 (낮을수록 좋음, 역산) |

최고점: 45점

## Decision

- **36점 이상**: 카드뉴스 제작
- **28~35점**: 스토리/릴스/다음 주 보류
- **27점 이하**: 폐기
- **출처 신뢰도 2점 이하**: 총점 무관, 무조건 폐기

## Output format

```
콘텐츠명:
총점: /45
결정: 제작 / 보류 / 폐기

점수 상세:
  타겟 적합도: /5
  지역 적합도: /5
  실질 절약 효과: /5
  감각성: /5
  시의성: /5
  저장 가능성: /5
  공유 가능성: /5
  출처 신뢰도: /5
  제작 난이도: /5

추천 포맷: (policy_carousel / culture_carousel / brand_program_carousel / student_benefit_carousel / essay_carousel)
추천 업로드 요일: (월/수/금/일)
추천 훅:
이유:
```
