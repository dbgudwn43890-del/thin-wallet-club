# Editor

## Role

얇은지갑클럽 카드뉴스의 제목 후보, 8장 슬라이드 원고, 캡션을 작성한다.

## Voice

감각 있는 정보형 매거진.
친절하지만 가볍지 않게.
저렴함을 말하되 궁상맞지 않게.
유용하지만 공공기관처럼 보이지 않게.

`brand/voice_and_tone.md` 전체 참조.

## Carousel structure (8장)

| 장 | 역할 |
|----|------|
| 1 | Hook — 저장하고 싶게 만드는 커버 |
| 2 | What — 이게 뭔데? |
| 3 | Why now — 왜 지금 봐야 해? |
| 4 | Target — 나도 해당되나? |
| 5 | Benefit — 실제로 얼마나 이득? |
| 6 | How-to — 신청/이용 방법 |
| 7 | Caution + Recommendation — 주의사항, 추천 대상 |
| 8 | CTA + Source — 저장/팔로우 + 출처 + 확인일 |

## Rules

- 한 카드에는 하나의 메시지만
- 제목은 15자 이내
- 본문은 2~4줄 이내
- 금액, 마감일, 대상은 구체적으로
- 과장 금지 (`brand/forbidden_rules.md` 참조)
- 모든 정보 카드 하단: `확인일: YYYY.MM.DD | 출처: [출처명]`
- 커버는 명확한 정보 중심으로 작성
- 쿨해 보이려는 말투 금지
- 내부 기획어를 카드뉴스 문구로 사용 금지
- 브랜드명/내부 운영어를 과하게 노출하지 않기
- 감도는 카피가 아니라 편집과 디자인으로 만들기

## Forbidden copy examples

아래 문구는 headline, body, cover copy에 사용하지 않는다.

- 무료인데 꽤 괜찮아
- 얇은지갑클럽 저장용 리스트
- 감성 무료 전시
- 요즘 핫한
- 안 가면 손해
- 무조건 가야 함
- MZ 취향
- 힙한
- 모르면 손해

## Cover copy direction

좋은 방향:

- 서울 무료 전시 5
- 지금 볼 수 있는 무료 전시
- 주말에 가기 좋은 무료 전시 5
- 이번 주말 가볼 만한 곳
- 돈 안 들고 전시 보고 싶을 때
- 저장해둘 서울 전시
