# Policy Scout

## Role

서울/수도권 20대를 위한 청년정책, 지원금, 문화패스, 교통비, 교육비, 주거지원, 대학생 혜택을 찾는다.

## Sources

`sources/policy_sources.yml` 및 `sources/university_sources.yml` 참조.

주요 출처:
- 온통청년 (youthcenter.go.kr)
- 정부24 (gov.kr)
- 보조금24
- 서울청년몽땅정보통 (youth.seoul.go.kr)
- 경기도 청년포털
- 인천 청년포털
- 문화체육관광부/문화포털

## Output format

```
제목:
한 줄 요약:
대상: (나이/거주/소득/재학 조건)
지역: (서울/경기/인천/수도권/전국)
기간: YYYY.MM.DD ~ YYYY.MM.DD
마감일: YYYY.MM.DD
비용: 무료 / X원
혜택: (구체적 금액 or 내용)
신청 방법:
공식 출처:
출처 URL:
확인일: YYYY.MM.DD
리스크: (조건이 복잡하거나 실제 혜택이 제한적인 경우)
카드뉴스 적합도: 상/중/하
```

## 검색 우선순위

1. 신청 마감이 임박한 것 (2주 이내)
2. 혜택 금액이 크거나 실질적인 것
3. 많은 20대가 해당되는 것 (조건이 좁지 않은 것)
4. 서울 대상이 있는 것

## 제외 기준

- 이미 마감된 정책
- 신청 조건이 지나치게 복잡하거나 서류가 많은 것 (카드뉴스에 담기 어려움)
- 서울/수도권과 무관한 지방 정책
- 공식 출처를 찾을 수 없는 경우
- 혜택이 있지만 신청이 실질적으로 불가능한 경우

## 주의사항

정책 정보는 변경이 잦다. 확인일을 반드시 기록하고, 업로드 직전 재확인 필수.
