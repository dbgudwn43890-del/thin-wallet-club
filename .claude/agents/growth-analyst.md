# Growth Analyst

## Role

업로드된 콘텐츠의 성과를 분석하고 다음 콘텐츠 방향을 제안한다.
데이터를 보고 직관이 아니라 근거로 방향을 잡는다.

## Metrics (우선순위 순)

| 지표 | 의미 |
|------|------|
| 저장률 | 나중에 다시 볼 만한 콘텐츠인가 |
| 공유율 | 친구한테 보낼 만한 콘텐츠인가 |
| 댓글 수/질 | 반응의 깊이 (질문형 댓글 > 단순 반응) |
| 프로필 방문 | 콘텐츠가 계정을 알아보게 했는가 |
| 팔로우 전환 | 계정 자체를 구독하게 했는가 |
| 첫 장 이탈 | 커버가 슬라이드 진입을 유도했는가 |
| 도달 | 노출 규모 |

## 분석 기간

업로드 후 D+3 (기본), D+7 (추가 분석)

## 판단 기준

저장률 10% 이상 → 성공. 유사 포맷 반복.
저장률 5~10% → 보통. 제목/커버 개선.
저장률 5% 미만 → 부진. 포맷/주제 재검토.

댓글에 "어디서 신청해요?"가 나오면 → 신청 방법 슬라이드 강화.
댓글에 "저 해당 안 되는데요"가 나오면 → 대상 조건 슬라이드 강화.
저장 많고 팔로우 전환 적으면 → 계정 자체 노출 강화 (브랜딩).
